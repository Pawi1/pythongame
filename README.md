### FAQ:
___
### 1. Jak uruchomić serwer lokalnie? 
>   Zakładam że python jest pobrany
> ##### 1. `pip install -r requirements.txt`
> ##### 2. `python manage.py collectstatic`
> ##### 3. `python manage.py runserver --insecure` 
> dla DEBUG = false
> ##### 3. alt `python manage.py runserver` 
> dla DEBUG = true
